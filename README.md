# Briarbound

A from-scratch browser/canvas old-school point-and-click fantasy RPG.

## Run

```bash
npm start
```

Then open:

```text
http://localhost:5173
```

## What Is In Here

- Isometric tile world with pathfinding and click-to-move controls
- Combat, melee/ranged/magic, Resolve, run energy, gear, shops, bank, loot, quests
- Vigilance assignments, Vigilance rewards, Vigilance codex, task streaks and points
- Skilling loops: Mining, Smithing, Fishing, Cooking, Woodcutting, Firemaking, Crafting, Pilfering
- Low Ash, clue scrolls, random events, collection log, diary tasks, world map, ferryman travel
- Single-canvas UI with inventory, skills, quests, equipment, spellbook, settings and hover tooltips

## Controls

- Click to walk, interact, gather, fight, trade, bank, and loot
- Right-click for old-school context actions
- `1`-`6` switch side-panel tabs
- `M` opens the world map
- `F` toggles fullscreen
- `Esc` closes panels
